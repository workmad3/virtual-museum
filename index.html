Virtual Museum

Document physical artefacts as a community

Download this project as a .zip file Download this project as a tar.gz file

Why

I'm looking after the keys to a room of stuff from the very early days of computing. This collection of stuff is never going to make it into a museum setting. It really needs to be photographed and described. Knowledge of these artefacts is individually remembered, and a means is needed to gather together descriptions of the artefacts.

What

Virtual Museum is basically a wiki that is to be used to catalogue and describe related artefacts in a collection. Collection item descriptions can be developed by users, and comments can be used to add related information.

Contributors who don't feel like curating wiki content may prefer to use comments to add information to item descriptions, and later these comments might be gathered and incorporated into the wiki content by people who enjoy curating page content.

Features

Strikethrough is used to indicate features still to be implemented.

Contributors must have registered themselves as users, there is a class of admin users who can perform a few some privileged operations.

Pages and their comments support rich content: Text, hyperlinks to other pages, images, YouTube and Vimeo video.

As each page describes a museum artefact, each is identified with a catalogue or accession number.

Status

In development, with features as above.
